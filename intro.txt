                                               Events  

    JavaScript Events are actions that happen in the browser.Common events include mouse clicks, keyboard presses and form submissions.Event handlers are JavaScript functions that respond to these events, allowing developers to create interactive 
web applications.

 Events:

onclick	    The user clicks an HTML element
onmouseover	The user moves the mouse over an HTML element
onkeydown	The user pushes a keyboard key
onload	    The browser has finished loading the page
onsubmit     when a form is submitted
oncontextmenu The user right-clicks on an element ===
